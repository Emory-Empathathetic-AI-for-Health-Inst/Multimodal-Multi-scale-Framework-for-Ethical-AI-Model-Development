# Reconciliation Logs Directory

This directory contains audit logs and historical consolidation records retained for traceability.

## Files
- `source_map.md` — source-to-destination mapping records
- `move_log.md` — chronological move/consolidation operations
- `duplicate_conflicts.md` — duplicate/conflict resolution decisions
- `phase_completion.md` — phase completion and deferred-work snapshot

These files are intentionally preserved as process records and may reference historical source provenance.
