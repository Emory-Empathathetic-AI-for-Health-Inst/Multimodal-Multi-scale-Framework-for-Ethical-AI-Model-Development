# Gap Analysis Directory

This section tracks implementation coverage and prioritization for the MEFINDER tooling/model roadmap and is colocated under `reconciliation/` for centralized project governance.

## Files
- `tooling_overview_status.md` — current implemented/partial/missing status by tool area
- `missing_tools_priority.md` — prioritized backlog of missing capabilities and suggested starting points

## Status conventions
- Tool status: `Implemented`, `Partial`, `Missing`
- Maturity labels (when used): `Unknown`, `Present`, `Runnable`, `Reproducible`

## How to maintain
- Update evidence paths when repository structure changes
- Reassess status when new capabilities are added or validated
- Keep entries tied to observable artifacts in this repository
